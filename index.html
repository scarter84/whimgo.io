Federated Nodes

Each device operates autonomously. Lose connection in a tunnel — the vehicle keeps running on its local model.

Tailscale Mesh

WireGuard-based direct connections with automatic NAT traversal. VPS tunnel fallback for anywhere-access.

Zero Cloud Dependency

All LLM inference runs on local hardware via Ollama. Your data never leaves your network.

Modular Tabs

Build your own stack. Enable only the tabs you need. Share community modules. Keep it lightweight.

// component registry

The Ecosystem

Every piece is independent, every piece connects. Pick what you need, ignore the rest.

Whim Terminal Desktop

The command center. An 18-tab Tkinter application with LLM model routing, voice engine, APRS/HAM radio, geofencing, audio capture, device management, and a full settings panel. Cross-platform: Linux, macOS, Windows.

Whim.m Mobile

Android companion app served as a 5-tab WebView. Voice recording, file library, AI chat, wake word detection, and cross-device messaging. Connects via VPS tunnel or Tailscale with auto-reconnect.

Whim.v Vehicle

Vehicle dashboard interface for a 13.6" Android head unit. Runs a local Qwen 1.5B model for offline autonomy. Handles OpenClaw commands, telemetry, and priority-based task delegation.

OpenClaw Protocol

The AI gateway protocol. JSON-RPC over WebSocket connecting the terminal to Ollama inference. Supports model switching, context injection, and multi-model routing (DeepSeek for logic, Llama for creative, Qwen for tools).

Voice Engine Audio

Full audio pipeline: XTTS v2 text-to-speech with persona voice clones, wake word detection (openWakeWord / Porcupine), live spectrogram (Whim-Scope), gain/HPF/AGC/EQ tuning for noisy environments.

GeoF Location

Geofencing and location awareness engine. Integrates with APRS packet radio for station tracking, supports custom geofence zones with trigger actions and notifications.

HAM / Doppler Telemetry

Amateur radio integration with Doppler weather radar overlay. APRS raw packet decoding, station list, signal activity visualization, and weather intelligence.

WISP Agent

Whim's Integrated Systems Processor. The resident AI agent for system operations, diagnostics, vault browsing, and automated task execution across the mesh.

Enoch / OpenClaw Bot Discord

Discord bot with the MeritClaw progression system. 10 badge tiers that unlock real capabilities: memory vaults, parallel tool execution, voice synthesis, image generation, and proactive assistance.

Get Started

Clone it. Configure it. Run it. No accounts, no cloud keys, no telemetry.

whim-terminal — bash
# Clone the repository
$ git clone https://github.com/scarter84/whim-terminal.git
$ cd whim-terminal

# Set up the environment
$ python3 -m venv venv
$ venv/bin/pip install -r requirements.txt

# Install Ollama (local LLM inference)
$ curl -fsSL https://ollama.ai/install.sh | sh
$ ollama pull deepseek-r1:32b

# Launch
$ venv/bin/python openclaw_tkui.py
